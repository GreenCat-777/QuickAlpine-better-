# QuickAlpine(better)
A application that allows you to quickly, launch, remove, and install the latest available alpine for kindle.
# How is this "Better" than QuickAlpine?
Not really (some even consider it worse) but it instead downloads the newer alpine kindle which was named Alpine(better)

# How to setup.
1. Extract the zip from the latest release
2. drag the documents folder to kindle. (if asked to merge folders select yes)


# What to use for.
Examples of use cases.
Installing Latest Alpine Easily.
Uninstalling Alpine Easily.
Running Alpine Easily
